# Privacy Policy

Effective date: March 26, 2026

Slop Watch is a Chrome extension that helps users vote on whether YouTube videos and channels appear AI-generated. This Privacy Policy explains what information is processed, how it is used, and your choices.

## 1. Information We Collect

We collect only the data needed for extension features:

- Extension settings stored locally in your browser (for example: enabled state, threshold settings).
- A randomly generated install identifier (UUID) stored locally, used to associate your votes with your install.
- Voting data you submit (for example: AI or human vote on a video or channel).
- YouTube identifiers needed for voting and scoring:
  - Video IDs
  - Channel handles
- Optional username, only if you choose to set one for leaderboard display.

## 2. Information We Do Not Collect

We do not intentionally collect:

- Health information
- Financial or payment information
- Authentication secrets (passwords, PINs, security answers)
- Personal communications (emails, messages, chats)
- Precise location (GPS)

## 3. How We Use Data

We use collected data to:

- Store your preferences locally.
- Record and update votes.
- Calculate aggregate AI-score percentages for videos/channels.
- Show user stats and leaderboard rankings.
- Maintain extension functionality and reliability.

## 4. Data Storage and Sharing

- Local browser storage is used for settings and install UUID.
- Backend data is stored in Firebase Realtime Database to support scoring and leaderboard features.
- Data is not sold.
- Data is not used for advertising.

Note: Like most online services, infrastructure providers may process technical metadata (such as IP address) for security, logging, and service operations.

## 5. Legal Basis and User Control

You control whether to:

- Use the extension
- Submit votes
- Set a username for leaderboard display

You may stop data collection by uninstalling the extension.  
You may request deletion of optional username-linked data by contacting us.

## 6. Data Retention

We retain data as long as needed to operate core features (voting aggregates, stats, leaderboard) or until deletion is requested where applicable.

## 7. Security

We use reasonable measures to protect stored and transmitted data. No system can be guaranteed 100% secure.

## 8. Children’s Privacy

Slop Watch is not directed to children under 13, and we do not knowingly collect personal data from children.

## 9. Changes to This Policy

We may update this Privacy Policy from time to time. Updates will be posted at the same public URL with a revised effective date.
